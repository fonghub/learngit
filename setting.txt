# 禁止换行符自动转换
git config --global core.autocrlf false
# 设置Git仓库username
git config --global user.name "your name"
# 设置Git仓库email
git config --global user.email "your email"

# 三个概念
1.工作区：
2.暂存区：
3.版本库：

三个区的修改对比：
工作区  vs  版本库  =  git diff 被修改文件名
暂存区  vs  版本库  =  git diff --cached  被修改文件名

# 查看提交历史
1.git log  查看有效的提交历史
2.git reflog  查看所有的提交历史

# 撤销/回退
1.撤销工作区修改
使用版本库文件覆盖工作区文件：git checkout -- filename

2.撤销暂存区修改（到工作区）
使用版本库五年级覆盖暂存区文件：git reset HEAD filename

3.回退版本库
回退到上一个版本库：git reset HEAD^
回退到制定comm_id版本库：git reset HEAD comm_id

# 删除文件
1.本地删除：rm -rf filename
2.版本库删除：git rm filename
3.生产提交comm_id：git commit -m "descript"

# 分支管理

1.创建分支:git branch branchName
2.切换分支:git checkout branchName
3.合并分支:git merge branchName
4.删除分支:git branch -d branchName
5.查看分支:git branch
6.创建并切换分支:git checkout -b branchName
